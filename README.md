# Bank-Churn-Dashboard
 📊 Project Overview
This project analyzes a dataset of 500 bank customers to identify factors contributing to customer churn.  
The goal is to uncover patterns in customer demographics, product usage, and account behavior that influence whether a customer stays with or leaves the bank.

 🗂 Dataset Description
The dataset includes the following variables:
- CustomerID: Unique identifier for each customer  
- Surname: Customer’s last name  
- CreditScore: Creditworthiness score (350–850)  
- Geography: Country of residence (France, Spain, Germany)  
- Gender: Male or Female  
- Age: Customer’s age  
- Tenure: Years with the bank  
- Balance: Account balance  
- NumOfProducts: Number of banking products used  
- HasCrCard: Credit card ownership (1 = Yes, 0 = No)  
- IsActiveMember: Active membership status (1 = Active, 0 = Inactive)  
- EstimatedSalary: Estimated salary  
- Churn: Whether the customer left the bank (1 = Yes, 0 = No)  

 🎯 Assignment Objectives
- Use different tools to analyze customer churn.  
- Identify key factors influencing churn (age, tenure, products, geography, etc.).  
- Provide insights into customer behavior and retention strategies.  

🛠 Tools & Skills
 - Microsoft Excel: Pivot Tables, charts, conditional formatting, data cleaning.  
 - Power BI: Interactive dashboards, DAX measures, slicers.  
 - SQL: Querying and preprocessing structured data.  
 - Python (Pandas, Matplotlib, Seaborn): Data cleaning, statistical analysis, visualization.  
 - Data Analysis & Storytelling: Translating raw data into actionable business insights.

 ✨ Features
- Pivot Table analysis of churn by - age group, tenure, geography, and product usage.  
- Visualizations highlighting churn trends and customer demographics.  
- KPI metrics such as average credit score, salary, and churn rate.  
- Comparative analysis of churned vs retained customers.  

🌟 Highlights
- Discovered that senior customers and those with 3 products show higher churn rates.  
- Identified churn spikes at 5 and 10 years of tenure, suggesting critical retention points.  
- Found that churn patterns vary across France, Spain, and Germany, with notable differences in customer behavior.  

 
 📈 Expected Insights
- Which age groups are most likely to churn  
- Impact of tenure and product usage on churn  
- Differences in churn across countries and genders  
- Financial indicators (balance, salary) linked to churn behavior
